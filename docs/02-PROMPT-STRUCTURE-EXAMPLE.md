# PROMPT STRUCTURE EXAMPLE

**Purpose:** This shows HOW to structure prompts, NOT WHAT to build.

**What to learn from this:**
- Phase tagging with emojis (🟧 Foundation, 🔸 Polish, 🔶 Functionality)
- Casual student voice ("Keep it basic please", "Ok, let's improve")
- Flexible phase ordering (Foundation → Polish → Functionality, not the default order)
- Progressive prompt structure (basic → visual → functional → debug)
- Intentional pedagogical bugs (Prompt 4 = debugging exercise)
- Completion checklist + badge gamification
- A-ha moment celebration

**What NOT to do:**
- ❌ Don't copy Wordle-specific content (grids, keyboards, word games)
- ❌ Don't use this as a template for all projects
- ✅ DO use this to understand the teaching flow and prompt patterns

# Project #2: Wordle Clone

## 🟧 OBJECTIVE
Make a word guessing game, similar to wordle

## 🟧 FOUNDATION

**Prompt 1: Build the Basic Game Layout**
> "Let's create the popular word-guessing game called Wordle.
We need:
5X6 boxes for the letters
Keyboard under this
Prompt Pilot Wordle Game as the title,
Keep it basic please
Add animations to the letters

## 🔸 POLISH

**Prompt 2: Make It Look Professional**
> "Ok, let's improve the visuals and make the site look more professional
The target market is educated professionals
Add a header with soft gradient
Add a footer with social media handles for X
Add a how to play section for beginners
Use a cleaner, modern font and bump the size slightly for readability.

## 🔶 FUNCTIONALITY

**Prompt 3: Make the Keyboard Work & Add Colors**
> "The keyboard doesn't function, let's intergrate it
Allow users to type the letters in and use backspace.
Add logic to show errors for less than 5 letters, invalid words, duplicates etc

Add tile flipping animation and color feedback: green for correct position, yellow for correct letter in wrong position, gray for letters not in the word. Include winning messages based on score"

**Prompt 4: Fix the Word List Bug**
> "Moved is not in the word list. Please debug and ensure we have a comprehensive list for user entries"

## 🎁 Bonus Prompts (Optional)

**Play Again Button**
> "Add a Play Again button that appears after winning or losing so players can start a new game."

**Remove the Cheat Code**
> "Remove the console.log that shows the secret word so players can't cheat."

**Hard Mode**
> "Add a hard mode toggle where players must use revealed hints in subsequent guesses."

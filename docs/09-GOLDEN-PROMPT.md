# 🌟 Golden Prompt (Optional - Don't Force It)

## What
A prompt where the learner restates their understanding and asks "Is my understanding correct?" - demonstrates authentic learning through asking.

## When to Use

### ✅ Use When:
- Project has a key conceptual breakthrough moment
- Understanding the "why" matters as much as the "what"
- Fits naturally into the learning flow

### ❌ Don't Use When:
- It feels artificial or forced
- The concept is straightforward
- It interrupts the flow

## Structure (if using)

```
[Explain concept in simple language]. Is my understanding correct?

[Explain related concept/implication]. Is my understanding correct?

[Tie it to the a-ha moment]. Can we do this?
```

## Key Traits

- Zero jargon, child-friendly language
- Shows authentic questioning process
- Usually appears at the project's main a-ha moment
- Reinforces understanding before building

## Example

> "So when I fetch data from an API, I'm asking another computer for information and it sends back a response in JSON format. Is my understanding correct?
>
> This means I can pull live data from anywhere - weather, stocks, crypto prices - as long as they have an API. Is my understanding correct?
>
> So I could build a dashboard that shows REAL prices updating automatically! Can we do this?"

## Remember

**Only use if it genuinely helps the learning. Many projects work fine without it.**
